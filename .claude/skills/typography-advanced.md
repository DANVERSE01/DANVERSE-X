# Typography Advanced

## Variable Font Motion
- Animate `font-variation-settings` with GSAP
- Axis ranges: `wght 100->900`, `wdth 75->125`, `slnt 0->-15`

## Cinematic Text Treatments
- SVG text clip masks for image/video-through-letterforms
- ScrambleText controls: charset, speed, delimiter options
- Kinetic typography: each character as an independent GSAP target

## Modern CSS Layout + Type
- CSS Anchor Positioning for tooltip/popover without Floating UI
- Fluid type scale: `clamp(1rem, 2.5vw + 0.5rem, 3rem)`
- Container queries with `@container` and `cqi` units
- `:has()` for parent-state logic without JavaScript

## CSS Architecture
- `@layer` ordering: reset / base / components / utilities
- Native CSS nesting patterns: `&:hover`, `& + &`, nested `@media`
